용어정리

1. extern
- 함수일때 :
	함수에 사용될때 extern을 사용하면 다른 함수에서 해당 함수를 호출할 수 있다. (default 방식)
	extern 지정은 함수가 정의된 화일외에 다른 화일에서 함수의 호출을 허용한다.

- 변수일때
	extern 변수는 함수 밖에서 선언되어 선언된 시점이후 모든 함수에서 사용가능한 변수이다.
	각 함수간에 자료를 공유하는 경우에 사용되는 변수이다.
	ex) A 파일에 int erp 선언
		B, C 파일에 erp 변수 사용하고 싶을 시
		B, C 파일에 extern int erp 라고 해서 사용한다.

2. static
	: 정적함수
	- 같은 이름을 가진 함수를 파일마다 만들 수 있다.
	- 각 파일에서 선언 되었을 때만 사용 할 수 있다.

3. inline
	: 일반 함수와는 달리 함수를 호출 하지 않고 함수를 사용하는 부분에 함수의 코드를 복제해 넣어준다.
	- 함수 호출 과정이 없어 속도는 빠르지만, 
	  함수의 코드가 복제되어 실행 파일의 크기가 커진다.
